Maintenance App Details

This Fuel Station Maintenance App, developed for Sapetrol Company, is a comprehensive solution designed for managing and monitoring the operations of a fuel station. Built using React and Ionic, the application seamlessly integrates with an API to handle tasks such as uploading images and videos, which are then accessible on an admin dashboard. It features a login system, various operational pages, and real-time reporting capabilities.

Accomplishments

  • API Integration: Enabled efficient communication with the server for data exchange and media uploads.
  • Media Uploads: Implemented functionality for users to upload images and videos directly to the API.
  • Admin Dashboard: Provided comprehensive access to maintenance reports and media for streamlined management.
  • Authentication: Built a secure login page for user access control.
  • Multi-Page Design: Developed multiple operational pages tailored to specific maintenance and inspection needs.
  • User-Friendly Interface: Leveraged React Ionic to deliver a responsive and intuitive UI.

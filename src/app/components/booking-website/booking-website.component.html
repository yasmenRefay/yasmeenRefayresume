Booking Website Details

A comprehensive ticket booking website consisting of 15 pages, designed to facilitate event ticket reservations. The platform includes an admin dashboard for full control, a login page for secure access, and features such as daily reports, shifts management, discounts, and coupon tracking. The website supports both light and dark modes for user customization. Developed using HTML, CSS, and Bootstrap for a responsive and user-friendly design.

Accomplishments

  • Designed and implemented a responsive 15-page website optimized for seamless user experience.
  • Developed a feature-rich admin dashboard for event and user management.
  • Integrated daily report generation and shift management functionality.
  • Added advanced features like discount handling and coupon management.
  • Ensured a dynamic user interface with light and dark mode options.
  • Built the project using clean, modular HTML, CSS, and Bootstrap for maintainability and scalability.
